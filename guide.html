Arcade Pit Files

There are two zip files below that you need.

The first is an emulator folder with all the files necessary to run Snes9x (1.53), Fusion, and FCE Ultra 2.2.2. Set them up however you like, including your controller options.

The other file has the roms and states you need to be able to do the challenges. Drop everything into the same folder and you should be good to go. If you want to merge it with your own files, just make sure the paths work by trying to load any of the states (batman on the nes has a few you can try).


  • Dropbox Folder with current files.

You will need to stream them both emulators to me at 512x448 resolution and a bitrate of 1000 kbps. Note that The NES resolution is 512x480, so FCE Ultra will need to be blackboxed a bit. Also make sure MSPaint is ready. Note that Fusion, the genesis emulator, might require monitor capture. I don't know any way around this, it is some sort of Windows 7 issue.

Useful Emulator Keyboard Shortcuts

  • Snes 9x: F1-F12 load states 1-12

  • FCEUX (NES): Number keys choose the state and 'p' loads the state

  • Fusion (GEN): Choose the State with F6/F7 and load the state with F8... CAUTION: don't hit F9 (saves over the state)
